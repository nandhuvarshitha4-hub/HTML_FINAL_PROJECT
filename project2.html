CSS
CSS (Cascading Style Sheets) is used to style and design web pages.It allows developers to manage layout, colors, fonts, spacing, animations, and responsive design. CSS supports concepts like selectors, box model, flexbox, grid, and media queries to create modern, device-friendly websites. It controls layout, colors, fonts, spacing, and responsiveness of HTML elements. CSS helps make websites visually attractive and user-friendly.
javascript
JavaScript is a programming language used to make web pages interactive and dynamic. It handles actions like button clicks, form validation, animations, and data updates. JavaScript works with HTML and CSS to create responsive and interactive websites
